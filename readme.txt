== Description ==

Plugin to display how may clicks (access) a post or page has.

== Installation ==

Follow the below steps to intall the plugin:
1- Send plugins's folder to plugins path, it should be /wp-content/plugins/
2- Access the blog panel and activate the plugin
3- Now, in the Settings area, a new link will appear "Clicks Counter", click on that link to go to plugin options page
4- Change the options you want and click on button Save Changes

To start showing the form you must use the chartag [show_clicks], put it right on the place you want the cliks appears.
